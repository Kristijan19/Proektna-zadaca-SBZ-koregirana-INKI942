# Систем за препорака на производи
**Knowledge-Based Recommendation System**  
Проектна задача — Системи базирани на знаење 2026

## Структура
```
main.py             # Влезна точка
knowledge_base.py   # База на знаење (seed data, категории, намени)
database.py         # SQLite слој (CRUD операции)
inference_engine.py # Инференциски мотор (правила за филтрирање)
gui.py              # PyQt5 графички интерфејс
requirements.txt    # Потребни библиотеки
```

## Инсталација
```
pip install -r requirements.txt
```

## Стартување
```
python main.py
```

## Опис
Апликацијата содржи три модули:
- **Препорака** — филтрирање по категорија, намена, буџет и рејтинг
- **База** — додавање и бришење на производи
- **Статистика** — преглед на податоците
